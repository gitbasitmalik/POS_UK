# AuraFlow POS — Enterprise Edition

A high-performance, modern Point of Sale system built for efficiency, accuracy, and premium user experience.

## 🚀 Features

- **Checkout & Self-Checkout**: Dual-mode operational interface.
- **Inventory Management**: Real-time stock tracking with automated reordering logic.
- **Supplier Manager**: Full procurement lifecycle from draft to delivery.
- **Promotions & Meal Deals**: Dynamic bundle pricing and multi-buy logic.
- **Yellow Label Manager**: AI-powered markdown pricing to minimize waste.
- **Gift Card Registry**: Corporate and customer gift card management.
- **Audit Trail**: Secure logging of all administrative actions.
- **Multi-Language Support**: Enterprise-ready localization (10+ languages).
- **Offline Mode**: Resilience for intermittent connectivity.

## 🛠️ Tech Stack

- **Framework**: React 18 with Vite
- **Styling**: Vanilla CSS (High Performance)
- **State Management**: Zustand (Persistent)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Type Safety**: TypeScript

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 🏗️ Project Structure

- `/src/pages`: Main application screens (Suppliers, Inventory, Checkout, etc.)
- `/src/components`: Reusable UI components and global modals.
- `/src/store.ts`: Central state management and business logic.
- `/src/types.ts`: Comprehensive type definitions.
